Массивы и указатели в С++

Массивы из структур

struct Rectangle
{
    int length;
    int width;
};
Rectangle rects[4]; // объявляем массив с 4-ма прямоугольниками
Присвоим ему значение элементу массива с индексом 0:
rects[0].length = 15;

Индекс массива

Перечисление в качестве индекса.
Массивы принимают в качестве индекса только целые числа, поэтому присвоить индексу тип перечисления не получется:
testScores[StudentNames::JOHN] = 65; // Вот так будет ошибка!!!
Но перечисления можно преобразовать в целое число оператором static_cast:
testScores[static_cast <int>(StudentNames::JOHN)] = 65;
Это то же неудобно, поэтому можно положить структуру в пространство имен:
namespace StudentNames
{
    enum StudentNames
    {
        SMITH, // 0
        ANDREW, // 1
        IVAN, // 2
        JOHN, // 3
        ANTON, // 4
        MISHA, // 5
        MAX_STUDENTS // 6
    };
}
 
int main()
{
    testScores[StudentNames::JOHN] = 65;
}


enum Animals
{
    ANIMAL_CAT = 3
};
array[ANIMAL_CAT] = 5;
При объявлении массива индекс должен быть константой типа compile-time (которая определяется во время компиляции). 
Используем макрос:
#define ARRAY_WIDTH 4
int array[ARRAY_WIDTH];

Перечисление используется, чтобы определить к какому наименованию относится индекс массива.
enum StudentNames
{
    SMITH, // 0
    ANDREW, // 1
    IVAN, // 2
    JOHN, // 3
    ANTON, // 4
    MAX_STUDENTS // 5
};
 
int main()
{
    int testScores[MAX_STUDENTS]; // всего 5 студентов
    testScores[JOHN] = 65;
 
    return 0;
}


Используем символьную константу:
const int arrayWidth = 7;
int array[arrayWidth];

А вот так делать нельзя, так как данная переменная не является константой.
int temp = 8;
const int width = temp;
int array[width];

Так тоже ошибка:
int width;
std::cin >> width;
int array[width];

Инициализация массивов

Список инициализаторов.
int array[5] = { 4, 5, 8, 9, 12 }; 

Инициализация всех элементов нулем:
int array[5] = { };

uniform-инициализация, без знака "=":
int array[5] { 4, 5, 8, 9, 12 };

Передача массива в функцию

Чтобы функция не изменяла значения элементов массива, переданного в неё в качестве параметра, то нужно сделать массив константным:
void passArray(const int array[5])
{
    array[0] = 11; // Oшибка!!!
}

Размер массива

Если передать оператору sizeof(array) массив, то он вернет размер всех элементов массива.
А чтобы узнать сколько элементов в массиве, нужно размер массива в байтах поделить на размер типа sizeof(int).
