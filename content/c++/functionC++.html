Полезные функции С++

std::swap()

Заголовочный файл <algorithm>, для C++11 используется заголовок <utility>

Меняет местами значение двух переменных.
	int a = 3;
    int b = 5;
	std::swap(a, b); // меняем местами значения переменных a и b

std::sort()

Заголовочный файл <algorithm>

Сортирует значения по возрастанию.
sort (<итератор начала сортировки>, <итератор конца сортировки >);
int sort_int_array[] = {5,74,11,65,77,84,13,53};
 vector<int> sort_int_vector (sort_int_array, sort_int_array+8);
